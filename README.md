# game-project
CODECRACK - Code Breaking Game
CODECRACK is an interactive logic-based puzzle game built with vanilla JavaScript, HTML5, and CSS3. The goal of the game is to crack a hidden 4-digit code using logic and feedback provided after each attempt.

🚀 Features
Two Difficulty Levels: * Beginner: Symbols in the code are unique (no repetitions).

Advanced: Symbols can repeat, adding a significant layer of complexity.

Time Challenge: A countdown timer (4 minutes) for advanced players to increase the stakes.

Interactive UI: Drag-and-drop mechanics for selecting and placing code symbols.

User Management: Simple Sign-up and Login system using localStorage.

Responsive Feedback: Visual and audio cues for correct/incorrect guesses and game events.

Dynamic Storytelling: An animated typewriter effect for the game instructions.

🛠️ Technologies Used
Frontend: HTML5, CSS3 (including Flexbox/Grid for layout).

Logic: Vanilla JavaScript (ES6+).

Storage: Browser localStorage for persisting user data.

Multimedia: HTML5 Audio API for sound effects.

🎮 How to Play
Login/Sign-up: Create a user to start playing.

Choose Level: Select between "Beginner" and "Advancing" from the menu.

Crack the Code: Drag the symbols into the empty slots and click "Send".

Feedback:

Well (White): Correct symbol in the correct position.

Good (Black): Correct symbol but in the wrong position.

Win/Loss: Crack the code within 10 attempts (and within the time limit for advanced mode) to win!
