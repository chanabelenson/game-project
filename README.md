CODECRACK - Code Breaking Game

CODECRACK is an interactive logic-based puzzle game built with vanilla JavaScript, HTML5, and CSS3. The goal of the game is to crack a hidden 4-digit code using logic and feedback provided after each attempt.

This project represents an early stage in my development journey, focusing on core programming concepts, DOM manipulation, and interactive user experience design.

🚀 Features

Two Difficulty Levels:
- Beginner: Symbols in the code are unique (no repetitions).
- Advanced: Symbols can repeat, adding a significant layer of complexity.

Time Challenge:
A countdown timer (4 minutes) for advanced players to increase the stakes.

Interactive UI:
Drag-and-drop mechanics for selecting and placing code symbols.

User Management:
Simple sign-up and login system using localStorage.

Responsive Feedback:
Visual and audio cues for correct/incorrect guesses and game events.

Dynamic Storytelling:
Animated typewriter effect for game instructions.

🛠️ Technologies Used

Frontend: HTML5, CSS3 (Flexbox/Grid layout)
Logic: Vanilla JavaScript (ES6+)
Storage: Browser localStorage
Multimedia: HTML5 Audio API

🎮 How to Play

Login/Sign-up:
Create a user to start playing.

Choose Level:
Select between "Beginner" and "Advanced".

Crack the Code:
Drag the symbols into the empty slots and click "Send".

Feedback:
- Well (White): Correct symbol in correct position
- Good (Black): Correct symbol but wrong position
- Win/Loss: Solve within attempts (and time limit in advanced mode)
